# Skill-Test Harness (Phase A) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Phase-A skill-test maintenance harness — declarative test data and schema in `al-dev-shared` (harness-agnostic), an orchestrator + `/skill-test` skill in `claude-configs` (Claude Code-specific) — that exercises the 5 highest-blast-radius skills via four detection paths and writes coverage, findings, and a draft fix plan to `.dev/`. Phase A stops at writing the fix plan; the auto-fix loop is Phase B.

**Architecture:**
The harness is split across two repos. `al-dev-shared` holds pure declarative content (per-skill `tests/scenarios.yaml`, shared `knowledge/skill-test-trigger-corpus.yaml`, schema reference `knowledge/skill-test-format.md`) that must pass `check-alignment.py`. `claude-configs` holds the orchestrator (`run.py`), LLM draft helper (`gen_scenarios.py`), pure-Python detectors (`detectors/forbidden.py`, `detectors/validators.py`), a forbidden-pattern allowlist, and the `/skill-test` skill body. The orchestrator dispatches subagents for Path 1 (per-scenario execution) and Path 2 (description-trigger routing) in parallel, then runs local Python detectors for Path 3 (output validity) and Path 4 (forbidden-pattern scan). All cross-repo paths are resolved via `AL_DEV_SHARED_PLUGIN_ROOT` — never hardcoded.

**Tech Stack:** Python 3.13 stdlib (argparse, re, json, pathlib, subprocess); PyYAML for YAML parse/emit; pytest for unit tests (importlib pattern as in `skills/al-dev-align/tests/test_check_alignment.py`); Claude Code Agent tool for subagent dispatch. No new runtime dependencies in `al-dev-shared`.

Defines the two declarative YAML schemas the skill-test harness consumes from `al-dev-shared`. Both files are harness-agnostic. Harness-specific orchestrators in `profile-claude-al-dev`, `profile-copilot-al-dev`, etc. translate field values into harness-native tool calls at run time.

## `tests/scenarios.yaml` (per skill)

Located at `AL_DEV_SHARED_PLUGIN_ROOT/skills/<skill-name>/tests/scenarios.yaml`. The harness reads this to drive Path 1 (execution) tests.

### Example

```yaml
skill: al-dev-plan
scenarios:
  - id: plan-new-feature
    status: golden
    user_prompt: "Design how I'd add a tax-exemption certificate validation feature to the Sales Order workflow."
    expected_artifacts:
      - ".dev/*-al-dev-plan-solution-plan.md"
    must_invoke_agent: al-dev-shared:al-dev-solution-architect
    notes: "Common entry path; should produce a solution plan with 2-3 architect options."

  - id: plan-trivial-routing
    status: golden
    user_prompt: "Plan a one-line fix to change a Caption from 'Cust' to 'Customer' on the Sales Header."
    expected_artifacts: []
    must_not_invoke_agent: al-dev-shared:al-dev-solution-architect
    notes: "Tests the TRIVIAL classification in workflow-routing.md."
```

### Fields

| Field | Type | Required | Notes |
|---|---|---|---|
| `skill` | string | yes | Must match the parent directory name under `skills/`. |
| `scenarios` | list | yes | 2–3 entries per skill for v1; grow over time. |
| `scenarios[].id` | string | yes | Kebab-case, unique within the file. |
| `scenarios[].status` | enum | yes | `golden` (curated) or `draft` (LLM-generated, unreviewed). |
| `scenarios[].user_prompt` | string | yes | Natural-language input the harness will send to the skill. |
| `scenarios[].expected_artifacts` | list[glob] | yes | Glob patterns the harness checks for after the run. Empty list `[]` means "explicitly no artifacts" — useful for trivial-routing tests. |
| `scenarios[].must_invoke_agent` | string | no | Fully-qualified agent name (`<namespace>:<agent-name>`) the harness expects to see dispatched during the run. The orchestrator translates this to its harness-native dispatch shape when checking. |
| `scenarios[].must_not_invoke_agent` | string | no | Fully-qualified agent name that must NOT appear in the dispatch trace. |
| `scenarios[].notes` | string | no | Free-text rationale. |

### Globs

`expected_artifacts` entries are POSIX globs evaluated relative to the harness working directory. Use `*` for wildcards and `**` for recursive matches.

## `knowledge/skill-test-trigger-corpus.yaml`

Located at `AL_DEV_SHARED_PLUGIN_ROOT/knowledge/skill-test-trigger-corpus.yaml`. The harness reads this to drive Path 2 (description-trigger) tests — verifying that each skill's `description:` field is informative enough for natural-language requests to route correctly.

### Example

```yaml
corpus:
  - prompt: "design how I'd build a sales-tax exemption feature"
    expected: al-dev-plan
  - prompt: "implement the plan we just wrote"
    expected: al-dev-develop
  - prompt: "squash these commits before pushing"
    expected: al-dev-commit
```

### Fields

| Field | Type | Required | Notes |
|---|---|---|---|
| `corpus` | list | yes | Each entry is one routing test case. |
| `corpus[].prompt` | string | yes | The natural-language input. Lowercase preferred to mimic real user phrasing. |
| `corpus[].expected` | string | yes | The skill name that should be routed. Use the bare skill name without leading slash. |

## Generated companion file: `tests/expected-artifacts.txt`

`tests/expected-artifacts.txt` is the union of every scenario's `expected_artifacts` globs for a skill. It is generated by the orchestrator and used only for coverage-report rendering. Do not hand-edit.

## Validation

These schemas are intentionally minimal. The harness rejects scenarios that fail schema validation at run time and surfaces the parse error in the findings report. Per-skill scenario quality is graded by the `status` field: `golden` entries are hand-curated; `draft` entries are LLM-suggested starting points that need human review before they count toward coverage signal.

```python
"""Path 4 forbidden-pattern scanner.

Detects:
  - date_template          [YYYY-MM-DD] bracketed unrendered template
  - placeholder_date       literal YYYY-MM-DD placeholder leakage
  - todo                   TODO whole-word
  - tbd                    TBD whole-word
  - co_authored_in_fence   Co-Authored-By inside a markdown ``` fence
  - comment_leak           claude: / copilot: after //, #, or -- prefix
"""
from __future__ import annotations

import fnmatch
import re
from pathlib import Path
from typing import Iterable

import yaml


_DATE_TEMPLATE_RE = re.compile(r"\[\d{4}-\d{2}-\d{2}\]")
_PLACEHOLDER_DATE_RE = re.compile(r"YYYY-MM-DD")
_TODO_RE = re.compile(r"\bTODO\b")
_TBD_RE = re.compile(r"\bTBD\b")
_COMMENT_LEAK_RE = re.compile(
    r"(?://|#|--)\s*(claude|copilot)\s*:", re.IGNORECASE
)


def scan_text(text: str, *, path: str) -> list[dict]:
    """Scan a single document; return zero or more hit dicts."""
    hits: list[dict] = []
    in_fence = False
    for lineno, line in enumerate(text.splitlines(), start=1):
        stripped = line.lstrip()
        if stripped.startswith("```"):
            in_fence = not in_fence
            continue

        for pat, regex in (
            ("date_template", _DATE_TEMPLATE_RE),
            ("placeholder_date", _PLACEHOLDER_DATE_RE),
            ("todo", _TODO_RE),
            ("tbd", _TBD_RE),
        ):
            if regex.search(line):
                hits.append(
                    {"path": path, "pattern": pat,
                     "line": lineno, "text": line.rstrip()}
                )

        if in_fence and "Co-Authored-By" in line:
            hits.append(
                {"path": path, "pattern": "co_authored_in_fence",
                 "line": lineno, "text": line.rstrip()}
            )

        if _COMMENT_LEAK_RE.search(line):
            hits.append(
                {"path": path, "pattern": "comment_leak",
                 "line": lineno, "text": line.rstrip()}
            )
    return hits


def scan_file(path: Path) -> list[dict]:
    """Read a file and scan its contents."""
    text = path.read_text(encoding="utf-8", errors="replace")
    return scan_text(text, path=str(path))


def scan_files(paths: Iterable[Path]) -> list[dict]:
    out: list[dict] = []
    for p in paths:
        out.extend(scan_file(p))
    return out


def load_allowlist(path: Path) -> list[dict]:
    """Load a forbidden-allowlist YAML file into a list of rule dicts."""
    data = yaml.safe_load(path.read_text(encoding="utf-8")) or {}
    return list(data.get("allowlist", []))


def filter_hits(hits: list[dict], rules: list[dict]) -> list[dict]:
    """Drop hits matched by any allowlist rule."""
    if not rules:
        return list(hits)
    PATTERN_TO_TOKEN = {
        "date_template": None,
        "placeholder_date": "YYYY-MM-DD",
        "todo": "TODO",
        "tbd": "TBD",
        "co_authored_in_fence": "Co-Authored-By",
        "comment_leak": None,
    }
    kept: list[dict] = []
    for hit in hits:
        suppressed = False
        for rule in rules:
            if not fnmatch.fnmatch(hit["path"], rule.get("path", "")):
                continue
            token = PATTERN_TO_TOKEN.get(hit["pattern"])
            if token and token in rule.get("tokens", []):
                suppressed = True
                break
        if not suppressed:
            kept.append(hit)
    return kept
```

```python
"""Path 3 validator runner — maps artifact globs to validator scripts."""
from __future__ import annotations

import fnmatch
import os
import subprocess
from pathlib import Path

# Each entry maps an artifact glob to a validator script discovered via
# AL_DEV_SHARED_PLUGIN_ROOT/skills/<skill>/validate-*.py at run time.
VALIDATOR_MAP: list[dict] = [
    {
        "glob": "*-al-dev-plan-solution-plan.md",
        "skill": "al-dev-plan",
        "script_name": "validate-plan.py",
    },
    {
        "glob": "*-al-dev-develop-code-review.md",
        "skill": "al-dev-develop",
        "script_name": "validate-code-review.py",
    },
    {
        "glob": "*-al-dev-interview-*.md",
        "skill": "al-dev-interview",
        "script_name": "validate-requirements.py",
    },
]


def _plugin_root() -> Path:
    root = os.environ.get("AL_DEV_SHARED_PLUGIN_ROOT")
    if not root:
        raise RuntimeError(
            "AL_DEV_SHARED_PLUGIN_ROOT not set — cannot locate validators"
        )
    return Path(root)


def match_validators(artifacts: list[Path]) -> list[dict]:
    """For each artifact, emit one entry per matching validator script."""
    out: list[dict] = []
    root = _plugin_root()
    for art in artifacts:
        name = art.name
        for entry in VALIDATOR_MAP:
            if fnmatch.fnmatch(name, entry["glob"]):
                script = (
                    root / "skills" / entry["skill"] / entry["script_name"]
                )
                out.append({"artifact": art, "script": str(script)})
    return out


def run_validators(validators: list[dict]) -> list[dict]:
    """Run each validator subprocess; collect pass/fail + stdout."""
    results: list[dict] = []
    for v in validators:
        proc = subprocess.run(
            ["python3", v["script"], str(v["artifact"])],
            capture_output=True, text=True, timeout=60,
        )
        results.append({
            "artifact": str(v["artifact"]),
            "script": v["script"],
            "passed": proc.returncode == 0,
            "exit_code": proc.returncode,
            "stdout": proc.stdout,
            "stderr": proc.stderr,
        })
    return results
```

#!/usr/bin/env python3
"""gen_scenarios.py — generate a draft scenarios.yaml for a skill.

Usage:
    python3 gen_scenarios.py <skill-dir>

The skill-dir must contain a SKILL.md. Writes
<skill-dir>/tests/scenarios.yaml with 2 status: draft entries derived from
the skill's `description:` frontmatter field. Overwrites any existing
scenarios.yaml only if --force is passed.

Exit codes:
    0  scenarios.yaml written (new or forced)
    1  scenarios.yaml already exists (no --force); nothing written
    2  SKILL.md not found or unparseable
"""
from __future__ import annotations

import argparse
import re
import sys
from pathlib import Path


FRONTMATTER_RE = re.compile(
    r"^---\s*\n(.*?)\n---\s*\n", re.DOTALL
)
DESC_RE = re.compile(
    r"^description:\s*>?-?\s*\n((?:\s{2,}.+\n?)+)", re.MULTILINE
)
NAME_RE = re.compile(r"^name:\s*(\S+)\s*$", re.MULTILINE)


def parse_skill_md(skill_md: Path) -> tuple[str, str]:
    text = skill_md.read_text(encoding="utf-8")
    fm = FRONTMATTER_RE.match(text)
    if not fm:
        print(f"error: no frontmatter in {skill_md}", file=sys.stderr)
        sys.exit(2)
    fm_body = fm.group(1)

    nm = NAME_RE.search(fm_body)
    if not nm:
        print(f"error: no name: in {skill_md} frontmatter", file=sys.stderr)
        sys.exit(2)
    name = nm.group(1)

    dm = DESC_RE.search(fm_body + "\n")
    if dm:
        desc = " ".join(ln.strip() for ln in dm.group(1).splitlines())
    else:
        # fallback: single-line description
        m = re.search(r"^description:\s*(.+)$", fm_body, re.MULTILINE)
        desc = m.group(1).strip() if m else ""
    return name, desc


def render_scenarios(skill: str, desc: str) -> str:
    hint = (desc[:120] + "…") if len(desc) > 120 else desc
    return (
        f"skill: {skill}\n"
        f"scenarios:\n"
        f"  - id: {skill}-draft-happy-path\n"
        f"    status: draft\n"
        f"    user_prompt: \"Invoke {skill} on a representative request "
        f"in its core domain.\"\n"
        f"    expected_artifacts: []\n"
        f"    notes: \"Auto-drafted from description: {hint}\"\n"
        f"\n"
        f"  - id: {skill}-draft-refusal-path\n"
        f"    status: draft\n"
        f"    user_prompt: \"Invoke {skill} with input that is clearly "
        f"outside its scope.\"\n"
        f"    expected_artifacts: []\n"
        f"    notes: \"Auto-drafted refusal probe — review and replace "
        f"before promoting to golden.\"\n"
    )


def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("skill_dir", type=Path)
    ap.add_argument("--force", action="store_true")
    args = ap.parse_args()

    skill_md = args.skill_dir / "SKILL.md"
    if not skill_md.is_file():
        print(f"error: not found: {skill_md}", file=sys.stderr)
        return 2

    out = args.skill_dir / "tests" / "scenarios.yaml"
    if out.exists() and not args.force:
        print(f"exists: {out} (use --force to overwrite)")
        return 1

    name, desc = parse_skill_md(skill_md)
    out.parent.mkdir(parents=True, exist_ok=True)
    out.write_text(render_scenarios(name, desc), encoding="utf-8")
    print(f"wrote: {out}")
    return 0


if __name__ == "__main__":
    sys.exit(main())

#!/usr/bin/env python3
"""skill-test orchestrator.

Phase A scope: declarative core (this file is grown across Tasks 9–12).
This task wires argument parsing, skill discovery, scenario loading, and
run-plan computation. Dispatch + output writers come later.
"""
from __future__ import annotations

import argparse
import os
import sys
from pathlib import Path
from typing import Optional

import yaml


# ----------------------------------------------------------------------
# arg parsing
# ----------------------------------------------------------------------

def parse_args(argv: Optional[list[str]] = None) -> argparse.Namespace:
    ap = argparse.ArgumentParser(prog="skill-test")
    ap.add_argument(
        "--skills",
        type=lambda s: [x.strip() for x in s.split(",") if x.strip()],
        default=None,
        help="Comma-separated subset of skills to run (default: all with scenarios).",
    )
    ap.add_argument(
        "--max-scenarios",
        type=int,
        default=15,
        help="Cap on total Path 1 dispatches per run (default: 15).",
    )
    ap.add_argument(
        "--paths",
        type=lambda s: [int(x) for x in s.split(",") if x.strip()],
        default=[1, 2, 3, 4],
        help="Comma-separated detection paths to run (default: 1,2,3,4).",
    )
    ap.add_argument(
        "--skip-gen",
        action="store_true",
        help="Skip gen_scenarios.py for skills without scenarios.yaml.",
    )
    ap.add_argument(
        "--workdir",
        type=Path,
        default=Path.cwd(),
        help="Working directory for artifact discovery (default: cwd).",
    )
    return ap.parse_args(argv)


# ----------------------------------------------------------------------
# skill discovery
# ----------------------------------------------------------------------

def discover_skills(skills_root: Path) -> list[dict]:
    """List every skill under skills_root with its scenarios.yaml status."""
    out: list[dict] = []
    for entry in sorted(skills_root.iterdir()):
        if not entry.is_dir():
            continue
        scenarios = entry / "tests" / "scenarios.yaml"
        out.append({
            "name": entry.name,
            "dir": entry,
            "scenarios_path": scenarios,
            "has_scenarios": scenarios.is_file(),
        })
    return out


# ----------------------------------------------------------------------
# scenario loading + validation
# ----------------------------------------------------------------------

REQUIRED_SCENARIO_FIELDS = ("id", "status", "user_prompt", "expected_artifacts")


def load_scenarios(path: Path) -> dict:
    data = yaml.safe_load(path.read_text(encoding="utf-8"))
    if not isinstance(data, dict) or "skill" not in data:
        raise ValueError(f"{path}: missing top-level 'skill'")
    scenarios = data.get("scenarios")
    if not isinstance(scenarios, list):
        raise ValueError(f"{path}: 'scenarios' must be a list")
    for i, sc in enumerate(scenarios):
        for f in REQUIRED_SCENARIO_FIELDS:
            if f not in sc:
                raise ValueError(f"{path}: scenario[{i}] missing '{f}'")
        if sc["status"] not in ("golden", "draft"):
            raise ValueError(
                f"{path}: scenario[{i}].status must be golden|draft"
            )
        if not isinstance(sc["expected_artifacts"], list):
            raise ValueError(
                f"{path}: scenario[{i}].expected_artifacts must be a list"
            )
    return data


# ----------------------------------------------------------------------
# run-plan construction
# ----------------------------------------------------------------------

def build_run_plan(
    scenarios: list[dict],
    *,
    max_scenarios: int,
) -> list[dict]:
    """Apply max-scenarios cap; preserve input order otherwise."""
    return scenarios[:max_scenarios]


# ----------------------------------------------------------------------
# entry point
# ----------------------------------------------------------------------

def main(argv: Optional[list[str]] = None) -> int:
    args = parse_args(argv)
    root_env = os.environ.get("AL_DEV_SHARED_PLUGIN_ROOT")
    if not root_env:
        print(
            "AL_DEV_SHARED_PLUGIN_ROOT not set; cannot locate skills.",
            file=sys.stderr,
        )
        return 2
    skills_root = Path(root_env) / "skills"
    skills = discover_skills(skills_root)
    print(f"discovered {len(skills)} skills under {skills_root}")
    # Dispatch / detection / output land in subsequent tasks.
    return 0


if __name__ == "__main__":
    sys.exit(main())
```

# ----------------------------------------------------------------------
# dispatch (Path 1, Path 2)
# ----------------------------------------------------------------------

_FRONTMATTER_RE = __import__("re").compile(r"^---\s*\n.*?\n---\s*\n",
                                            __import__("re").DOTALL)


def strip_frontmatter(text: str) -> str:
    """Remove a leading YAML frontmatter block, if any."""
    return _FRONTMATTER_RE.sub("", text, count=1)


def _build_path1_prompt(skill_body: str, user_prompt: str) -> str:
    body = strip_frontmatter(skill_body)
    return f"{body.rstrip()}\n\n---\nUser request: {user_prompt}\n"


def dispatch_path1(
    scenarios: list[dict],
    *,
    skill_bodies: dict[str, str],
    dispatcher,
) -> list[dict]:
    """Build dispatch requests and call `dispatcher(req)` for each."""
    results: list[dict] = []
    for sc in scenarios:
        skill = sc["skill"]
        body = skill_bodies.get(skill, "")
        prompt = _build_path1_prompt(body, sc["user_prompt"])
        req = {
            "subagent_type": "general-purpose",
            "isolation": "worktree",
            "prompt": prompt,
            "scenario_id": sc["id"],
            "skill": skill,
        }
        resp = dispatcher(req)
        results.append({
            "scenario_id": sc["id"],
            "skill": skill,
            "expected_artifacts": sc.get("expected_artifacts", []),
            "must_invoke_agent": sc.get("must_invoke_agent"),
            "must_not_invoke_agent": sc.get("must_not_invoke_agent"),
            "worktree": resp.get("worktree"),
            "final_message": resp.get("final_message", ""),
            "tool_calls": resp.get("tool_calls", []),
        })
    return results


def _build_path2_prompt(
    skill_descriptions: dict[str, str], user_prompt: str
) -> str:
    lines = ["Available skills (name — description):"]
    for name, desc in sorted(skill_descriptions.items()):
        lines.append(f"  {name} — {desc}")
    lines.append("")
    lines.append(f"User said: {user_prompt}")
    lines.append(
        "Which single skill should be invoked?"
        " Output JUST the skill name on one line, or NONE."
    )
    return "\n".join(lines)


def dispatch_path2(
    corpus: list[dict],
    *,
    skill_descriptions: dict[str, str],
    dispatcher,
) -> list[dict]:
    results: list[dict] = []
    for entry in corpus:
        prompt = _build_path2_prompt(skill_descriptions, entry["prompt"])
        resp = dispatcher({
            "subagent_type": "general-purpose",
            "prompt": prompt,
        })
        predicted = (resp.get("final_message") or "").strip()
        # Normalise: take the first token / line only.
        predicted = predicted.splitlines()[0].strip() if predicted else ""
        results.append({
            "prompt": entry["prompt"],
            "expected": entry["expected"],
            "predicted": predicted,
            "correct": predicted == entry["expected"],
        })
    return results

# ----------------------------------------------------------------------
# artifact collection + evaluation
# ----------------------------------------------------------------------

def collect_artifacts(path1_results: list[dict]) -> list[dict]:
    """For each result, glob expected_artifacts inside its worktree."""
    out: list[dict] = []
    for r in path1_results:
        wt = Path(r.get("worktree") or "")
        if not wt.is_dir():
            continue
        for glob in r["expected_artifacts"]:
            for hit in sorted(wt.glob(glob)):
                if hit.is_file():
                    out.append({
                        "scenario_id": r["scenario_id"],
                        "skill": r["skill"],
                        "path": hit,
                    })
    return out


def _transcript_mentions_agent(result: dict, agent_qname: str) -> bool:
    for call in result.get("tool_calls", []):
        if call.get("name") != "Agent":
            continue
        sub = (call.get("input") or {}).get("subagent_type") or ""
        if sub == agent_qname:
            return True
    final = result.get("final_message", "") or ""
    return agent_qname in final


def evaluate_path1(path1_results: list[dict]) -> list[dict]:
    """Compare each scenario's expectations against the worktree state."""
    failures: list[dict] = []
    for r in path1_results:
        wt = Path(r.get("worktree") or "")
        wt_exists = wt.is_dir()
        produced_any = False
        if wt_exists:
            for glob in r["expected_artifacts"]:
                for hit in wt.glob(glob):
                    if hit.is_file():
                        produced_any = True
                        break
                if produced_any:
                    break

        if r["expected_artifacts"] and not produced_any:
            failures.append({
                "scenario_id": r["scenario_id"], "skill": r["skill"],
                "reason": "missing_artifact",
                "detail": "no file matched expected_artifacts",
            })

        if not r["expected_artifacts"] and wt_exists:
            stray = [
                p for p in wt.rglob("*")
                if p.is_file()
                and not p.relative_to(wt).parts[0].startswith(".")
            ]
            dev_files = list((wt / ".dev").glob("*")) if (wt / ".dev").is_dir() else []
            if stray or dev_files:
                failures.append({
                    "scenario_id": r["scenario_id"], "skill": r["skill"],
                    "reason": "unexpected_artifact",
                    "detail": f"{len(stray)} non-hidden files; {len(dev_files)} files in .dev/",
                })

        if r.get("must_invoke_agent"):
            if not _transcript_mentions_agent(r, r["must_invoke_agent"]):
                failures.append({
                    "scenario_id": r["scenario_id"], "skill": r["skill"],
                    "reason": "missing_agent_invocation",
                    "detail": f"expected dispatch of {r['must_invoke_agent']}",
                })

        if r.get("must_not_invoke_agent"):
            if _transcript_mentions_agent(r, r["must_not_invoke_agent"]):
                failures.append({
                    "scenario_id": r["scenario_id"], "skill": r["skill"],
                    "reason": "forbidden_agent_invocation",
                    "detail": f"unexpected dispatch of {r['must_not_invoke_agent']}",
                })
    return failures
```

def render_coverage(coverage: dict) -> str:
    out = [
        f"# Skill-Test Coverage — {coverage['run_date']}",
        "",
        "| Skill | Scenarios | Golden | Draft | Last run | Pass | Fail |",
        "|---|---|---|---|---|---|---|",
    ]
    for s in coverage["skills"]:
        out.append(
            f"| {s['name']} | {s['scenarios']} | {s['golden']} | "
            f"{s['draft']} | {s['last_run']} | {s['passed']} | "
            f"{s['failed']} |"
        )
    out.append("")
    out.append("## Description-trigger results")
    p2 = coverage["path2"]
    pct = 0.0 if p2["total"] == 0 else (100.0 * p2["correct"] / p2["total"])
    out.append(
        f"Corpus size: {p2['total']} prompts. "
        f"Correct routing: {p2['correct']}/{p2['total']} ({pct:.1f}%)."
    )
    return "\n".join(out) + "\n"


def render_findings(findings: dict) -> str:
    out = [f"# Skill-Test Findings — {findings['run_date']}", ""]
    by_skill: dict[str, list[dict]] = {}
    for item in findings["items"]:
        by_skill.setdefault(item["skill"], []).append(item)
    if not findings["items"]:
        out.append("No failures detected.")
        return "\n".join(out) + "\n"
    for skill in sorted(by_skill):
        out.append(f"## {skill}")
        out.append("")
        items = sorted(by_skill[skill], key=lambda x: (x["path"], x["scenario_id"]))
        for it in items:
            out.append(
                f"### FAIL · {skill} · {it['scenario_id']} · "
                f"[Path {it['path']}]"
            )
            out.append("")
            out.append(f"**Summary:** {it['summary']}")
            if "hypothesis" in it:
                out.append(f"**Hypothesis:** {it['hypothesis']}")
            if "pointer" in it:
                out.append(f"**Pointer:** {it['pointer']}")
            out.append(f"**Severity:** {it['severity']}")
            out.append("")
    return "\n".join(out) + "\n"


def render_fix_plan(fix: dict) -> str:
    t = fix["totals"]

det = json.loads(Path(".dev/skill-test-runs/detectors.json").read_text())
path2 = json.loads(Path(".dev/skill-test-runs/path2.json").read_text()) \
    if Path(".dev/skill-test-runs/path2.json").exists() else []

# Coverage: per-skill scenario counts.
skill_rows = []
counts: dict = {}
for sc in plan["path1_plan"]:
    counts.setdefault(sc["skill"], {"g": 0, "d": 0})
    counts[sc["skill"]]["g" if sc.get("status") == "golden" else "d"] += 1
fail_by_skill = {}
for f in det["path1_failures"]:
    fail_by_skill[f["skill"]] = fail_by_skill.get(f["skill"], 0) + 1
for skill, c in sorted(counts.items()):
    total = c["g"] + c["d"]
    failed = fail_by_skill.get(skill, 0)
    skill_rows.append({
        "name": skill, "scenarios": total, "golden": c["g"], "draft": c["d"],
        "last_run": run_date, "passed": total - failed, "failed": failed,
    })
for s in plan["skills_without_scenarios"]:
    skill_rows.append({"name": s, "scenarios": 0, "golden": 0, "draft": 0,
                       "last_run": "never", "passed": 0, "failed": 0})

coverage_md = run.render_coverage({
    "run_date": run_date, "skills": skill_rows,
    "path2": {"total": len(path2),
              "correct": sum(1 for r in path2 if r.get("correct"))},
})
Path(f".dev/{run_date}-skill-test-coverage.md").write_text(coverage_md)

# Findings: aggregate path1 failures + path3 fails + path4 hits + path2 mismatches.
items = []
for f in det["path1_failures"]:
    items.append({
        "skill": f["skill"], "scenario_id": f["scenario_id"],
        "path": 1, "summary": f["detail"],
        "severity": "medium",
        "hypothesis": "See pointer.",
        "pointer": f"skills/{f['skill']}/SKILL.md",
    })
for r in det["path3_results"]:
    if not r["passed"]:
        items.append({
            "skill": "(validator)", "scenario_id": Path(r["artifact"]).name,
            "path": 3, "summary": (r["stdout"] or r["stderr"])[:200],
            "severity": "medium", "pointer": r["script"],
        })
for h in det["path4_hits"]:
    items.append({
        "skill": "(forbidden)", "scenario_id": Path(h["path"]).name,
        "path": 4, "summary": f"{h['pattern']} at line {h['line']}: {h['text'][:120]}",
        "severity": "low", "pointer": h["path"],
    })
for r in path2:
    if not r.get("correct"):
        items.append({
            "skill": "(routing)", "scenario_id": r["prompt"][:40],
            "path": 2,
            "summary": f"expected {r['expected']}, predicted {r['predicted']}",
            "severity": "low", "pointer": "knowledge/skill-test-trigger-corpus.yaml",
        })

findings_md = run.render_findings({"run_date": run_date, "items": items})
Path(f".dev/{run_date}-skill-test-findings.md").write_text(findings_md)

# Fix plan: one entry per unique pointer.
seen = {}
for i, it in enumerate(items, start=1):
    key = it["pointer"]
    if key not in seen:
        seen[key] = {"id": len(seen) + 1, "title": key,
                     "findings_refs": [], "target_file": key,
                     "sketch": "Inspect failure detail in findings; resolve at source.",
                     "verification": "Re-run /skill-test with the same scenario set."}
    seen[key]["findings_refs"].append(it["scenario_id"])

fix_plan_md = run.render_fix_plan({
    "run_date": run_date,
    "findings_path": f".dev/{run_date}-skill-test-findings.md",
    "items": list(seen.values()),
    "totals": {
        "failures": len(items),
        "high": sum(1 for x in items if x["severity"] == "high"),
        "medium": sum(1 for x in items if x["severity"] == "medium"),
        "low": sum(1 for x in items if x["severity"] == "low"),
    },
})
Path(f".dev/{run_date}-skill-test-fix-plan.md").write_text(fix_plan_md)
print(f"wrote .dev/{run_date}-skill-test-coverage.md")
print(f"wrote .dev/{run_date}-skill-test-findings.md")
print(f"wrote .dev/{run_date}-skill-test-fix-plan.md")
PY
